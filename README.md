# Minecraft Crafting Quiz

마인크래프트 조합법을 맞추는 Streamlit 퀴즈 앱입니다.

## 설정 방법
pip install -r requirements.txt
streamlit run app.py

## 테스트 계정
아이디: player
비밀번호: 1234

## 실행 방법
-퀴즈가 시작되면 자동으로 타이머가 시작됩니다. 모든 문제를 풀고 나면 총 소요 시간을 확인할 수 있습니다!
-인벤토리에 있는 아이템들을 드래그하거나 클릭으로 선택한 뒤 작업대 슬롯에 배치하여 바른 조합을 만드시면 됩니다!
